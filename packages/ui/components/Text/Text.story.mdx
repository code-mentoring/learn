There are a number of variant that we use for text, which can be set with the `variant` property.

    </Row>

    </Row>

      <Text variant="body1">body1</Text>
    </Row>
    <Row>
      <Text variant="body2">body2</Text>

    <Row>
      <Text variant="span">span</Text>
    </Row>
    <Row>

    </Row>
    <Row>
      <Text variant="strong">strong</Text>
    </Row>

<Text variant="title">title</Text>
<Text variant="subTitle">subTitle</Text>
