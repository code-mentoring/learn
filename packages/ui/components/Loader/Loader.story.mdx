---

## Center

By default the Loader is set to be centered within the parent container. This can be changed by setting the `center` property to `false`.

<Card>
    <Row>
        <Loader center={false} />
        <Loader />
    </Row>
</Card>

```tsx
<Loader center={false} />
<Loader />
```
