# Button

## Sizes
We use two sizes for our buttons; `small` and `large`. They can be set with the `size` property.
<Card>
    <Row>
      <Button>Button</Button>
    </Row>
    <Row>
      <Button size="large">Large Button</Button>
    </Row>
</Card>

```tsx
<Button>Button</Button>
<Button size="large">Large Button</Button>
```
