# Box

## Properties
There are three properties that can be used with the Box component: `padding`, `margin`, `shadow`. All three are optional.
Padding and Margin: it can be a `number` or a `Size` (from the theme). If not set, it will use the `md` size (1.6rem).

<Card>
  <Row>
    <Box shadow='main'>...content...</Box>
  </Row>
  <Row>
    <Box padding="lg" shadow='main'>...content...</Box>
  </Row>
  <Row>
    <Box margin="lg" shadow='main'>...content...</Box>
  </Row>
  <Row>
    <Box padding="lg" margin="lg" shadow='main'>...content...</Box>
  </Row>
</Card>

```tsx
<Box>...content...</Box>
<Box padding="lg">...content...</Box>
<Box margin="lg">...content...</Box>
<Box shadow="main">...content...</Box>
<Box padding="lg" margin="lg" shadow="main">...content...</Box>
```

## Shadows
It can be one of the theme's options (`main`, `success`, `error`). If not set, the box will have no shadows.

<Card>
  <Row>
    <Box>...content...</Box>
  </Row>
  <Row>
    <Box shadow='main'>...content...</Box>
  </Row>
  <Row>
    <Box shadow='success'>...content...</Box>
  </Row>
  <Row>
    <Box shadow='error'>...content...</Box>
  </Row>
</Card>

```tsx
<Box>...content...</Box>
<Box shadow='main'>...content...</Box>
<Box shadow='success'>...content...</Box>
<Box shadow='error'>...content...</Box>
```
