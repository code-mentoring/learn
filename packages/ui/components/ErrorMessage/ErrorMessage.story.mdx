# Text Alignment
There are two text alignment options for the Error Message; `center` and `right`. They can be set with the `center` property set to `true` 
or `false`. The `center` property, by default, will be set to `false`. Setting the `center` property to `true` will center the Error Message and setting the property to `false` will right align the Error Message.
